# youtube-app

# To run this program:

go to client folder: 

- Run -> npm start
- open in localhost:3000

go to server folder:

- Run -> node app.js / nodemon app.js
- server will run in port 4001

# Liblary Used

- ReactJS
- Semantic ui (CSS)
- Axios
- Express
- Socket.IO (for realtime both client and server)
